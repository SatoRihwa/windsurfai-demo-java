---

このワークフローは、シンプルな挨拶メッセージを表示するデモ用のワークフローです。

## 使い方

1. このワークフローを実行すると、現在の日付とともに挨拶メッセージが表示されます
2. オプションで名前を指定してパーソナライズされた挨拶を表示することもできます

## コマンド

```bash
# 基本的な使い方
/workflow hello-workflow

# 名前を指定する場合
/workflow hello-workflow --name=あなたの名前
```

## 例

```
こんにちは、世界！
今日は 2025年6月4日 です。
素晴らしい1日をお過ごしください！
```

## 注意点

- このワークフローはデモンストレーション用です
- 特別な依存関係は必要ありません
